Transaction Level eMulator (TLMu)

 TLMu is a modified version of QEMU that integrates with SystemC TLM-2.0 systems.

It is possible to create TLM-2.0 systems with multiple TLMu cores, of varying architectures. Each TLMu instance may provide a bare CPU core or a partial system with devices.

To clone the TLMu git:
% git clone git://github.com/edgarigl/tlmu.git

See the TLMu documentation for more info on how to build it and howto get started.

Developers should also visit the github page.

There will soon be a mailinglist available, in the meantime please contact edgar.iglesias at gmail.com or post on the github wiki if you have any questions.
