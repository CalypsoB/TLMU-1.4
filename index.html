Transaction Level eMulator (TLMu)

 TLMu is a modified version of QEMU that integrates with SystemC TLM-2.0 systems. Read more about TLMu here.

To check out TLMu:
% git clone git://github.com/edgarigl/tlmu.git

See the TLMu documentation for more info on how to build it and howto get started.

There will soon be a mailinglist available, in the meantime please contact edgar.iglesias at gmail.com if you have any questions.
